Features

🔀

REST → gRPC Translation

Gateway translates HTTP/JSON requests into typed gRPC calls. Proto files document the contracts.

🔐

JWT Authentication

HS256 Bearer token validation on every non-health endpoint. Tokens verified per-request, no session state.

⚡

Per-Client Rate Limiting

Token-bucket limiter keyed by IP (100 req/s, burst 200). Returns 429 when the bucket is empty.

📋

Structured Logging

Every request emits a JSON log line with method, path, status, latency, and request ID via Uber zap.

🐳

Docker + Compose

Single multi-stage Dockerfile; a build arg selects which binary to compile. make up starts all three.

⚙️

GitHub Actions CI

Automated vet, test, build all binaries, and golangci-lint runs on every push to main.

Live API Playground

All requests run against this live Vercel deployment. A demo token is generated automatically — no sign-up needed. Create a user first; the ID auto-fills subsequent forms.
